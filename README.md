# Quizgame

Laravel aplication for a quiz game type website.

Module available:
- Login
- Signup
- Admin Panel with CRUD operations for categories, questions and answers
- Contact messages 
- User roles (admin/user)
- Ajax paginations for categories
- front-end quiz
